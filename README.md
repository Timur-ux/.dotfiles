# Моя конфигурация Linux OS

Здесь лежат конфиги всех программ, которыми я пользуюсь:

- niri: оконный менеджер;
- dms: DankMaterialShell -- UI мультитул: строка состояния, экран блокировки, wallpapers и т.д.;
- kitty: терминал;
- neovim: лучший текстовый редактор;
- bash: настройка профиля.

## Установка

Для удобной установки используется утилита `stow`:

1. Перейти в домашнюю директорию

```bash
$ cd ~
```

2. Скопировать этот репозиторий

```bash
$ git clone --recurse-submodules https://github.com/Timur-ux/.dotfiles.git
```

3. Установить конфиги

```bash
$ stow .
```
